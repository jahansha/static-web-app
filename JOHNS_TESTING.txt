asked ChatGPT how to test the app in VSCode:
two ways:
Quickest Way
. Open your project folder in VSCode
. Navigate to your index.html
. Right-click → Reveal in File Explorer
. Double-click index.html

It will open in your default browser.

Recommended: Use VSCode Live Server (Best for Frontend Apps)
🔹 Install Live Server
1- Open VSCode
2- Go to Extensions (Ctrl+Shift+X)
3- Search for Live Server
4- Install the one by Ritwick Dey

5- Run Your App
6- Open your project folder
7-Right-click index.html
8-Click "Open with Live Server"
